# NovaLyrics 🎵✨ - by Shanudha Tirosh

NovaLyrics is a high-performance terminal-based lyrics visualizer designed to provide a stunning, synced musical experience. It features vibrant color transitions and smooth animations that bring your favorite songs to life right in your terminal.

## Description
This project is a rebranded and enhanced version of the original lyrics player, optimized for Shanudha Tirosh's portfolio. It uses Python's threading and Pygame's mixer to synchronize letter-by-letter animations with audio playback, creating a professional-grade CLI music experience.

## Features
- **Dynamic Synced Lyrics**: Precisely timed animations that match the audio rhythm.
- **Spectrum Color Transitions**: Cycles through a curated palette of vibrant ANSI colors.
- **Threaded Execution**: Seamlessly handles concurrent audio playback and visual effects.
- **Smart Pause Indicators**: Intuitive visual cues (...) during instrumental segments.
- **Cross-Platform Compatibility**: Fully functional on Windows, Linux, and macOS.
